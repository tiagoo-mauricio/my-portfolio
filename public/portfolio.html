PORTFÓLIO

Nesta seção, você encontrará uma seleção dos meus projetos recentes, cada um com sua própria história e abordagem única. Vamos dar uma breve olhada em alguns deles:

APLICATIVO DE COMPRAS ONLINE

DESCRIÇÃO: Este projeto envolve o design de um aplicativo de compras online user-friendly. O objetivo é criar uma experiência de compra online perfeita para os usuários, com fácil navegação, checkout rápido e suporte ao cliente eficiente.

DESCRIÇÃO TÉCNICA: O projeto foi desenvolvido utilizando princípios de design centrado no usuário. As ferramentas utilizadas incluem Sketch para wireframing e prototipagem, e InVision para testes de usabilidade.

WEBSITE DE RESERVAS DE HOTEL

DESCRIÇÃO: Este projeto envolve o design de um website intuitivo para reservas de hotel. O objetivo é facilitar o processo de reserva para os usuários, permitindo que eles encontrem e reservem quartos de hotel com facilidade.

DESCRIÇÃO TÉCNICA: O projeto foi desenvolvido utilizando princípios de design responsivo. As ferramentas utilizadas incluem Figma para design e prototipagem, e Adobe XD para testes interativos.

APLICATIVO MOVEL DE FITNESS

DESCRIÇÃO: Este projeto envolve o design de um aplicativo móvel interativo para fitness. O objetivo é motivar os usuários a manterem-se ativos e saudáveis através do rastreamento do progresso do exercício e da oferta de planos de treino personalizados.

DESCRIÇÃO TÉCNICA: O projeto foi desenvolvido utilizando princípios UX/UI modernos. As ferramentas utilizadas incluem Adobe XD para design e prototipagem, e Marvel App para testes interativos.
